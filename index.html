Custom woocommerce online shop

This was really interesting project because this store is nothing like any other Woocommerce store. Client had specific requests so I had to do many customizations in form of custom plugins that I had to write in order for this to work the way he wanted. I had many challenges during this project, like writing custom Mailing classes, writing backend for restaurant chiefs so they can answer orders in 30,45,60 minutes or cancel them, I implemented custom user blacklist , user order counting with awarding system. Also I've written custom order statistics plugin. Once all customizations were done, I had to configure Hetzner dedicated server to work as fast and secure as possible. I've implemented Redis caching, memcached as sessions cache, disk caching, software raid. Also I had to write custom backup script so both web files and databases are backing up 4 times a day on 3 offsite servers. After all that, I had to configure Cloudflare as CDN and caching so we can save some bandiwth and reduce server load. End result was great, client was very satisfied with speed and way his online ordering system works.
